## 외계어 사전

```java
class Solution {
    public int solution(String[] spell, String[] dic) {

        boolean check = false;
        for (String str : dic) {
            int count = 0;

            for (String word : spell) {
                if (str.contains(word)) {
                    count++;
                }
            }
            if (count == spell.length) {
                check = true;
                break;
            }
        }
        return check ? 1 : 2;

    }
}

```
